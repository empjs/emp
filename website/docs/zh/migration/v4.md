# v4 迁移要点

安装 v4 rc 包：

```bash
pnpm add -D @empjs/cli@rc @empjs/share@rc
```

按框架选择插件：

```bash
pnpm add -D @empjs/plugin-react@rc
pnpm add -D @empjs/plugin-vue3@rc
pnpm add -D @empjs/plugin-vue2@rc
```

## Rspack 2

保留 `defineConfig(...)` 结构，通过 `build.rspack` 暴露 Rspack 2 配置：

```ts
export default defineConfig({
  build: {
    target: 'es2018',
    rspack: {
      experiments: {
        css: true,
      },
    },
  },
})
```

## Module Federation 2

继续使用 `pluginRspackEmpShare(...)`，但发布前要重新验证 manifest、runtime 和类型声明。

## 不迁移范围

`@empjs/cdn-*` 和 `@empjs/lib-*` 保持独立版本线。`apps/**` 和 `website` 是示例、验收项目或站点，不进入自动发布范围。
