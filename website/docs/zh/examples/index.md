# 示例与验收

当前 v4 示例集中在 `apps/**`。这些应用既是使用样例，也是发布前真实验收面的来源。

| 类型 | 应用 |
| --- | --- |
| React Module Federation | `mf-host`、`mf-app` |
| Vue 2 Module Federation | `vue-2-base`、`vue-2-project` |
| Vue 3 Module Federation | `vue-3-base`、`vue-3-project` |
| 跨框架适配 | `adapter-host`、`adapter-app` |
| React 19 / Router / Tailwind 4 | `react-19-tanstack`、`tailwind-4` |
| Rspack 2 能力 | `rspack2-modern-module`、`rspack2-optimization` |
| 综合 demo | `demo` |

旧 `tailwind-2`、`tailwind-3`、`daisyui-demo`、`shadcn-ui` 等不属于当前 v4 apps 验收目标。
