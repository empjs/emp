# 插件生态

EMP 插件用于把框架、CSS、代码规范和构建能力接入同一套配置模型。当前 v4 主线覆盖：

- `@empjs/plugin-react`
- `@empjs/plugin-vue2`
- `@empjs/plugin-vue3`
- `@empjs/plugin-tailwindcss`
- `@empjs/plugin-postcss`
- `@empjs/plugin-lightningcss`
- `@empjs/plugin-stylus`
- `@empjs/biome-config`
- `@empjs/eslint-config-react`

v4 当前 Tailwind 主覆盖是 Tailwind CSS 4；旧 Tailwind 2 / 3 示例不再是当前 apps 验收目标。

## Agent-First 配置手册

完整插件配置示例维护在仓库 `skills/emp-v4-agent-first/references/plugins.md`。官网只列出当前插件面，具体接入步骤以仓库 Skill 为准。
