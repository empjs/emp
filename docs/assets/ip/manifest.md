# EMP Federation Fox IP Assets

This folder and the root `docs/assets` logo files define the EMP v4 project IP asset positions.

## Official Positions

| Position | File | Usage |
| --- | --- | --- |
| Official IP logo, high-resolution | `../emp-v4-readme-logo.png` | README hero logo and source image for future high-resolution usages. Transparent PNG, 1254 x 1254. |
| Official website/home logo | `../emp-federation-fox-full.png` | Rspress home hero image. Transparent PNG, 512 x 512. |
| Official navigation icon | `../emp-federation-fox-compact.png` | Rspress navbar/favicon-like small UI usage. Transparent PNG, 512 x 512. |
| Mascot dark showcase | `emp-federation-fox-mascot-dark.png` | Dark-background mascot/IP presentation image. Use for design previews, slides, and brand showcases. |
| Mascot launch scene | `emp-federation-fox-launch-scene.png` | Wide launch/release scene. Use for release banners, article covers, and social cards. |

## Rule

- Use `../emp-v4-readme-logo.png` as the canonical official IP logo source.
- Use `../emp-federation-fox-full.png` and `../emp-federation-fox-compact.png` for website runtime assets.
- Use the two `docs/assets/ip/*.png` images as visual materials, not as transparent UI icons.
- Do not replace the official EMP Federation Fox IP with the Rspack mascot or other project IP.
